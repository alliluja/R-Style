### Snippets:
- head     - Macro header
- macro    - head of MACRO procedure (private macro MacroName(var one): returnValue ...)
- while    - while loop
- db       - debugbreak operator
- for      - For (var i, 0, 10, 1)
- fora     - For (var item, itemlist) 
- if       - if (condition)
- note     - NOTE >>
- todo     - TODO:
- fixme    - FIXME:
- warn     - WARNING!!!
